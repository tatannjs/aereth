# aereth
petit jeu Action-aventure 3D, plateformer 

## Objectifs :
- consolider mes connaissances de bas niveau
- Apprendre les pipelines graphiques (OpenGl puis Vulkan)
- Construire un environnement 3D à partir de pas grand chose

## Speech :
Dans les cieux fracturés d’Aereth, des îles flottantes dérivent au-dessus d’un océan de nuages luminescents. Autrefois reliées par des courants d’énergie appelés Voies Aériennes, elles sont désormais instables depuis qu’un ancien Cœur céleste s’est brisé.
Les îles sont habitées par des Sylphins, de petites créatures bipèdes au design simple (corps rond, grands yeux, petites cornes ou oreilles flottantes). Elles canalisent l’énergie du vent et peuvent manipuler les éléments légers (air, électricité statique, poussière d’étoiles).
Quand les Voies Aériennes commencent à s’effondrer, les îles s’éloignent les unes des autres… et certaines chutent dans l’abîme.
